Reveil

Principe de fonctionnement

  • Après le choix du mode Reveil dans le menu, un nouveau terminal sera ouvert.
  • Dans ce dernier, vous devrez configurer une alarme en saissisant l'heure et la minute.
  • A l'heure choisie, une alarme sera activée

Chronomètre

Principe de fonctionnement

  • Après le choix du mode Chrono dans le menu, deux nouveaux terminaux sont ouverts.
  • Dans le terminal Chronomètre, vous pourrez récupérer jusqu'à 3 temps intermédiaires puis le temps final en pressant la lettre "t".
  • Dans le terminal Affichage Chronomètre, vous pourrez visualiser le temps depuis le lancement du chronomètre.

Minuteur

Principe de fonctionnement

  • Après le choix du mode Minuteur dans le menu, un nouveau terminal est ouvert.
  • Dans ce terminal Minuteur, vous paramétrez le temps souhaité pour votre minuteur. 
  • Puis un nouveau terminal Affichage minuteur est lancé pour visualiser le temsp restant.

Statistiques d'utilisation

Principe de fonctionnement

  • Après le choix du mode Statistiques dans le menu, deux choix sont disponibles.
  • En choissisant Afficher les statistiques, vous pouvez visualiser le nombre de chaque fonction lancer depuis le demarrage du Reveil_Berry.
  • Dans l'autre option, les compteurs seront remis à 0.

Monitoring

Principe de fonctionnement

  • En choissisant l'option monitoring du menu, vous lancer un nouveau terminal via un script bash.
  • Dans ce nouveau terminal, vous avez tous les "processus" en cours d'excution.
